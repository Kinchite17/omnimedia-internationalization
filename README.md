# omnimedia-internationalization
This website was established by the Internationalization Department of Zhejiang University Omnimedia to help students of Chu Kochen Honors College improve their English proficiency. It provides a platform for sharing learning resources, practical guidance, and international perspectives.

## Features

- PDF download entry on home page: `英语四六级高频词汇.pdf`
- Vocabulary quiz page with two modes:
  - English -> Chinese multiple choice (20 random questions per attempt)
  - Chinese -> English spelling (20 random questions per attempt, with letter-count hints)

## Vocabulary data

- Quiz data file: `data/vocabulary.json`
- Extraction and fallback note: `docs/vocabulary-data.md`
